#说明
Traivis: 部分收费的 CI server，收费标准与 github 类似，公开专案不收费，私人专案收费
jenkins: CI server, 免费，但是要自己架站

XCTool: 一套 FB 开发的测试用命令列工具，可以与 Travis, jenkins 整合

OCMock: 测试框架语言
Kiwi: 测试框架，使用 when/
